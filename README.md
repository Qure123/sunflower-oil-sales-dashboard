# sunflower-oil-sales-dashboard
Дашборд продаж растительного масла (Power BI) \
Анализ продаж по клиентам и номенклатуре за год. Визуализация сезонности, долей и ценовой динамики.
## Описание
Дашборд на основе агрегированных данных по 8 клиентам и 15 позициям растительного масла (подсолнечное, кукурузное; разные объёмы и бренды: БЛАГО, ДАРЫ КУБАНИ). 

Позволяет:
 - Выявлять топ-клиентов и ключевые товары по выручке и объёму;    
 - Анализировать сезонные колебания продаж;    
 - Сравнивать средневзвешенную цену за тонну между клиентами/товарами;    
 - Принимать управленческие решения по развитию прибыльных направлений.   
## Ключевые метрики
| **Метрика**                                      | **Описание / Формула**                          | **Примечание**                     |
|--------------------------------------------------|--------------------------------------------------|------------------------------------|
| **Total Revenue (Выручка)**                      | Суммарный доход компании за период (₽)          | Основной показатель продаж        |
| **Total Weight (Объём продаж)**                  | Общий объём проданного масла в тоннах           | Измеряется в тоннах               |
| **Avg Price per Ton (Средняя цена за тонну)**    | Выручка / Вес — средневзвешенная цена (₽/т)     | Корректно отражает ценовую политику |
| **Client Revenue Share %**                       | Доля клиента в общей выручке (%)                | Показывает вклад клиента          |
| **Weight Share %**                               | Доля клиента в общем объёме продаж (%)          | Анализ объёмной значимости        |
| **Non-Empty Months (Активные месяцы)**           | Количество месяцев с продажами у клиента        | Отражает регулярность спроса      |
###### Все визуализации связаны через фильтрацию: выбор клиента/товара автоматически обновляет остальные элементы — анализ «от общего к частному»
## Технические детали
 - **Инструмент**: Power BI Desktop (.pbix);
 - **Данные**: очистка данных проводилась уже в Power BI через язык Power Query.
   > Примечание: локальные расхождения (например, выручка > 0 при весе = 0 в одном месяце) были обнаружены при разведочном анализе и удалены на этапе подготовки данных перед построением панели мониторинга. Итоговые метрики рассчитаны на агрегированном уровне (по клиенту/товару/периоду), поэтому такие расхождения не влияют на результаты;
 - **Сортировка месяцев**: реализована через вспомогательный столбец MonthOrder (янв=1, фев=2, …, дек=12);
 - **Дизайн**: тематические элементы (подсолнух, голубь), краткие названия товаров, хронологическая упорядоченность.
